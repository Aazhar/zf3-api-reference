setServers()

Set servers

$servers can be an array list or a comma separated list of servers. One server in the list can be descripted as follows:

  • URI: [tcp://][:][?weight=]
  • Assoc: array('host' => [, 'port' => ][, 'weight' => ])
  • List: array([, ][, ])
